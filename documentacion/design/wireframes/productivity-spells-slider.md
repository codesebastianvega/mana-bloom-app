# Productivity Spells Slider Wireframe

`
+-------------------------------------------------------------+
| Hechizos de Productividad          [PRO] chip (derecha)     |
| copy helper / description                                     |
+-------------------------------------------------------------+
| <- slide wrapper ------------------------------------------>|
|  +------------------+   +---------------------------------+ |
|  | Chips (columna)  |   |   Main Spell Card (focus/pulso) | |
|  | - Profundo       |   |   - Title + icon                | |
|  | - Sprint         |   |   - Content (cristal / huevo)   | |
|  | - Micro          |   |   - Timer wrapper               | |
|  +------------------+   +---------------------------------+ |
|                                                             |
|  Control chips / helpers (auto, reiniciar, etc.)            |
+-------------------------------------------------------------+
| Pagination dots                                            |
+-------------------------------------------------------------+
`

## Secciones clave
1. **Header PRO**: título, chip PRO alineado a la derecha y descripción.
2. **Slide Wrapper**: Scroll horizontal con tarjetas; cada slide conserva la misma rejilla (chips + card principal).
3. **Card principal**: Espacio donde se renderiza el hechizo seleccionado (Cristal o Pulso).
4. **Controles secundarios**: chips reutilizables (auto, reiniciar, modo) justo debajo del timer.
5. **Dots**: indicadores de página alineados al centro.

Este wireframe sirve como plantilla para futuros hechizos: basta con sustituir el contenido del card principal y los textos/helpers específicos del hechizo.
